# IPv4 vs IPv6

## Basic Difference

| Feature | IPv4 | IPv6 |
|---|---|---|
| **Address Size** | 32-bit | 128-bit |
| **Format** | Decimal (e.g. 192.168.1.1) | Hexadecimal (e.g. 2001:0db8::1) |
| **Total Addresses** | ~4.3 billion | Bahut zyada (undecillion se bhi zyada) |
| **Header Complexity** | Complex header | Simplified header (fast processing) |
| **Security** | Optional (IPSec add karna padta hai) | Built-in support (IPSec) |
| **Configuration** | Manual ya DHCP | Auto-configuration support |

## IPv6 Kyun Zaroori Hua?

- IPv4 ke addresses **khatam ho rahe hain** (sirf ~4.3 billion hain, but duniya mein usse zyada devices hain)
- IoT devices (smart watches, sensors, appliances) ki wajah se demand bahut badh gayi hai unique addresses ki

> **Analogy**: IPv4 ek chhoti si housing society jaisa hai jisme limited houses (addresses) hain — full ho chuki hai. IPv6 ek bahut bada city hai jisme lagbhag unlimited houses ban sakte hain.

## IPv4 Address Example

```
192.168.1.1
```
4 octets, har ek 0-255 ke beech (8-bit each = 32-bit total)

## IPv6 Address Example

```
2001:0db8:85a3:0000:0000:8a2e:0370:7334
```
8 groups of hexadecimal numbers (16-bit each = 128-bit total)

## Viva One-Liners

- **Q: IPv4 aur IPv6 mein basic difference?**
 A: IPv4, 32-bit hai (~4.3 billion addresses), IPv6, 128-bit hai (bahut zyada addresses).
- **Q: IPv6 kyun laaya gaya?**
 A: Kyunki IPv4 ke addresses khatam ho rahe the, growing devices ki demand ke wajah se.
