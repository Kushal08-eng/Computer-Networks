## How These Notes Are Organized

- Files are numbered to preserve the tutorial's own learning sequence — from fundamentals through the OSI/TCP-IP layers to individual protocols.
- Each file includes definitions, comparison tables, and quick-reference points suited for revision and interview/viva preparation.
- Topics build on one another, so following the numerical order is recommended for a first read-through.

## Usage

These notes are intended for exam preparation, interview revision, and quick conceptual reference. Each file is self-contained and can be read independently once the foundational topics (1–9) are covered.
