## How These Notes Are Organized

- Files are numbered to preserve a logical learning sequence — from foundational concepts (network types, architecture) to layered models (OSI, TCP/IP) and individual protocols (TCP, UDP, HTTP, DNS).
- Each file includes definitions, comparison tables, and quick-reference points suited for revision and viva preparation.
- Topics build on one another, so following the numerical order is recommended for a first read-through.

## Usage

These notes are intended for exam preparation, interview revision, and quick conceptual reference. Each file is self-contained and can be read independently once the foundational topics (1–13) are covered.
