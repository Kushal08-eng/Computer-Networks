# Introduction to Computer Networks

## Computer Network Kya Hai?

Do ya zyada devices ka aapas mein connect hona taaki wo data share kar saken, resources share kar saken, aur communicate kar saken — isi ko **Computer Network** kehte hain.

> **Analogy**: Jaise ek city mein roads sabhi ghar aur offices ko connect karte hain taaki log aana-jaana aur saaman transport kar sakein — waise hi network cables/wireless signals devices ko connect karte hain taaki data transport ho sake.

## Ye Topic Kyun Padhna Zaroori Hai?

| Reason | Matlab |
|---|---|
| Internet samajhna | Jo bhi tum daily use karte ho (WhatsApp, YouTube, Gmail) — sab networking pe based hai |
| Backend/System Design | Software engineer banne ke liye networking fundamentals must hain |
| Interview | Networking basics (OSI, TCP/IP, HTTP) common interview questions hain |
| Debugging | Real projects mein network-level issues samajhne ke liye zaroori |

## Is Course/Notes Mein Kya Cover Hoga

- History — internet kaise start hua

- Protocols — HTTP, TCP, UDP, IP, DNS

- Devices — Router, Modem, Switch, Hub
- Security basics — NAT, Middleboxes

## Quick One-Liner

> "A computer network connects multiple devices to enable communication and resource sharing."
